# ImersaoDevAula03
Nesta aula 03 da imersão dev, programei um projeto que de certa forma diverte e interage com o usuário, onde o sistema escolhe um número de 0 a 10, e a pessoa tenta acerta que número é esse em apenas 3 tentativa, e caso erre ela recebe dicas simples para tentar acertar na próxima tentativa.
